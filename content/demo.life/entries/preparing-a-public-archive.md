Before publishing an archive, review the content the same way you would review a website.

Check the manifest, remove private notes, confirm image rights, and make sure collection pages only expose what you intend to share. A renderer can help with validation, but the archive owner remains in control of the source folder.

This project renders archive pages as server HTML so public content can be indexed by search engines.
